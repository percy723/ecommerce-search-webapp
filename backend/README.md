# Express backend  

## remember to add back from the \api the following components before running  
> node_modules  
> package.json  
> package-lock.json  
  
## run express  
`npm start`  
  
### npm dependencies
`npm express-generator`    
`npm install --save cors`  
`npm install request`  
`npm install https`  
`npm install mysql`  
`npm npm install yelp-fusion --save`  
`npm install bcrypt`
